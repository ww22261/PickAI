## 一、需求分析

### 1.1 产品定位
PickAI 是一款面向消费者的 AI 驱动智能选品工具，核心差异点是**AI动态生成风格问题**，而非固定模板。

### 1.2 核心功能
| 步骤 | 页面 | 功能描述 |
|------|------|----------|
| Step 1 | 需求输入页 | 用户自然语言描述购买需求 |
| Step 2 | 风格问题页 | AI动态生成2个精准问题 |
| Step 3 | 推荐结果页 | 展示风格总结+商品推荐 |

## 五、核心功能实现细节

### 5.1 AI动态问题生成

**实现方式**:
- 前端调用后端 `/api/ai/questions` 接口
- 后端调用火山引擎 DeepSeek-V3 API 生成针对性问题
- 失败时自动降级到品类匹配的本地问题库

**品类问题库**:
- 包类: 简约通勤 / 时尚轻熟 / 商务正式 / 休闲运动
- 衣服类: 宽松慵懒 / 修身合体 / 文艺复古 / 简约基础
- 耳机类: 音质优先 / 降噪功能 / 续航持久 / 外观颜值

### 5.2 拼多多商品搜索

**API调用流程**:
1. 用户输入关键词 + 选择风格 + 选择预算
2. 智能拼接搜索关键词（融入风格关键词）
3. 调用 `pdd.ddk.goods.search` API，带价格区间过滤
4. 返回真实商品数据（名称、价格、图片、销量等）
5. 二次价格过滤，确保商品价格严格在预算范围内
6. 智能评分排序后展示

**智能推荐实现**:
- 风格关键词映射：将用户选择的风格（如"简约低调"、"马卡龙色"）转换为电商搜索词
- 价格区间过滤：API层面设置 `min_normal_price` 和 `max_normal_price` 参数
- 二次价格校验：确保返回的商品价格严格在用户选择的预算范围内
- 品类识别：根据用户输入自动识别品类（包、衣服、耳机、帽子等）
- 动态商品名称：商品名称根据风格和品类智能生成

**预算区间配置**:
| 用户选择 | 价格范围（元） |
|---------|--------------|
| 25元以内 | 0.01 - 24 |
| 25-50元 | 26 - 49 |
| 50-150元 | 55 - 148 |
| 150-300元 | 155 - 298 |
| 300元以上 | 305+ |

**示例返回数据**:
```json
{
  "goods_id": 897379974566,
  "goods_name": "阿秋的店烟粉色通勤百搭内搭衬衫...",
  "goods_thumbnail_url": "https://img.pddpic.com/...",
  "min_group_price": 6000,
  "sales_tip": "916"
}
```

### 5.3 推广链接生成

**API**: `pdd.ddk.goods.promotion.url.generate`

**参数**:
- `p_id`: 推广位ID
- `goods_sign_list`: 商品签名列表

**返回示例**:
```
https://mobile.yangkeduo.com/duo_coupon_landing.html?goods_id=xxx&pid=xxx&cpsSign=xxx
```

### 5.4 商品评分模型

```
综合评分 = 风格匹配度(35%) + 性价比(30%) + 质量信号(20%) + 优惠力度(15%)
```
